# Funcionalidades
  - Cadastrar usuário
  - Login
  - Postagem filme
  - Editar filmes
  - Excluir filmes
  - Listar filmess

## Tecnologias utilizadas
### Backend
  - NodeJS/Express
  - MongoDB
### Mobile
  - React Native

# Como executar o projeto

## Pré-requitos: npm/yarn 

```bash
# clonar repositório  
git clone https://github.com/juniorsilvacc/app-netflix.git

# entrar na pasta do projeto mobile
cd app-netflix

# instalar dependências
yarn install

# executar projeto
yarn start
```
